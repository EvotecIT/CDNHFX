This code is here to support HtmlForgeX project. All files hosted here are available on JSDelivr CDN. It's so HtmlForgeX can use CDN capabilities for files that are big but are not hosted anywhere or for files that are modified from originals.

Information about how this works: https://www.jsdelivr.com/features

Project Code: https://www.jsdelivr.com/package/gh/evotecit/cdnhfx
